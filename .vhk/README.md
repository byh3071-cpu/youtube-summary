# Focus Feed VHK State

Repository VHK policy is defined in `RULES.md` and `docs/VHK_ADOPTION.md`.

- `ledger.jsonl` may be tracked as verification history.
- `context.md` and `brief.md` may be tracked when they are intentionally maintained.
- reports, incidents, backups, memory, refs, cloud metadata, and HARD_STOP are local-only.
